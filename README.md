# Calculadora

Calculadora com operações básicas em Python
Este projeto consiste em uma calculadora funcional desenvolvida em Python para realizar operações matemáticas básicas diretamente no terminal. 

O funcionamento da aplicação começa com a solicitação de dois números digitados pelo usuário, aceitando tanto valores inteiros quanto decimais. Em seguida, o sistema apresenta um menu para que uma das quatro operações principais seja escolhida: soma, subtração, multiplicação ou divisão. O processamento interno utiliza verificações condicionais para identificar a escolha e aplicar o operador matemático correspondente aos números fornecidos. Um detalhe importante na construção do código é a trava de segurança na operação de divisão, que impede que o programa tente dividir um número por zero, exibindo uma mensagem de erro apropriada em vez de interromper a execução.
